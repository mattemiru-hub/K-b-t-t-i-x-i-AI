# v1.0.1

Clarification-first release for `excel-dashboard-ai-skill`.

## Highlights

- adds a clarification gate to the skill workflow
- forces the AI to ask short targeted questions before guessing critical workbook mappings
- adds a dedicated Vietnamese clarification rules reference
- adds a Vietnamese repo-packaging playbook for building a complete skill repo in one pass

## What changed

- `SKILL.md`
  - added `Clarification Gate`
  - added `Required Question Style`
  - added explicit do-not-guess behavior for business-critical fields
- `references/clarification-rules-vi.md`
  - defines when the AI must stop and ask
  - defines which fields are too important to guess
  - defines suggested clarification questions
- `references/repo-packaging-playbook-vi.md`
  - documents the full process for packaging a complete repo from scratch
- `README.md` and `README_vi.md`
  - now explain the clarification-first behavior clearly

## Recommended use

Use this release when you want:

- the same premium Excel dashboard skill package as `v1.0.0`
- less field-guessing risk on unfamiliar workbooks
- a reusable repo structure for future Codex skill repos
