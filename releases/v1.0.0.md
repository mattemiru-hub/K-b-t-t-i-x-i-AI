# v1.0.0

First public-ready release of `excel-dashboard-ai-skill`.

## Highlights

- packaged as a shareable Codex skill repo
- includes styling automation for Excel dashboard polish
- includes preview export automation
- includes Vietnamese documentation and reusable prompt templates
- includes sample usage examples and sample output guidance

## Recommended use

Use this release when you want to:

- rebuild Excel files into refreshable executive dashboards
- keep the dashboard experience close to Power BI
- enforce PivotTable, PivotChart, Power Query, Power Pivot, and DAX based architecture
- share a repeatable skill package with other Codex users
